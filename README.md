## Description
gRPC with typescript using random api
- using node v12.16.0
- using gRPC v1.24.2

## Requirement
- protobuf

## Used Command
- command compile protobuf - ファイルを自動作成します。
```
$ sh generate.sh
```

## Usage
1. install protobuf by homebrew
```
$ brew install protobuf
```

2. install package
```
$ npm i 
```

3. server run
```
$ npm run server
```

4. client call
```
$ npm run client
```
