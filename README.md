# grpc-sample

## Description
Sample implementation of gRPC with typescript


## Requirement
- protobuf


## Usage
1. install protobuf by homebrew
```
$ brew install protobuf
```

2. install package
```
$ npm i 
```

3. compile protobuf
```
$ sh generate.sh
```

4. server run
```
$ npm start
```

5. client call
```
$ npm run client
```
